항상 배우고 끊임없는 도전과 성장을 추구하는 웹퍼블리셔

안녕하세요 웹퍼블리셔 김현기입니다.
저는 웹 사이트 제작을 할 때, 마크업언어와 다른 웹기술로 작업을 하면서 웹접근성과 웹표준을 항상 고려하고 새로운 웹 기술을 습득하면서 끊임없이 배우고 발전하고 있습니다.
아직 부족한 게 많지만 꾸준하게 노력하고 부족한 부분을 보완하면서 사용자의 편리성을 생각하는 웹퍼블리셔가 되겠습니다!

  • My SKILL

    제가 작업할때 사용한 언어들입니다.

  • HTML을 통해서 웹페이지의 구조를 설계하는 방법을 배웠습니다.

    CSS을 통해서 HTML의 구조적인 태그들을 활용하여 웹페이지를 꾸미고 시각적으로 표현하는 방법을 배웠습니다.
    그리고 다양한 디바이스(PC, 태블릿, 스마트폰)에 맞춰 화면 크기를 조정해 사용자 경험을 최적화하는 방법을 배웠습니다.

    JavaScript을 통해서 기본적인 function함수의 응용과 구조를 배웠습니다.
    그리고 웹페이지에 필요한 동적인 기능을 작성하는 방법을 배웠습니다.

    JQuery는 JavaScript 라이브러리로, jquery을 통해서 JavaScript의 기능을 더 쉽고 간단하게 사용할 수 있도록 만드는 방법을 배웠습니다.

    Illustration을 통해서 로고, 아이콘, 벡터이미지와 같은 이미지를 편집하는 작업을 합니다.

    Photoshop을 통해서 사진보정 및 편집하는 디자인 작업을 합니다.

    Figma를 통해서 웹사이트를 구현하기 전에 전체적인 UI 디자인을 설계하는 방법을 배웠습니다.

    GitHub를 통해서 작업했던 코드를 업로드하여
    관리하거나 프로젝트 파일을 공유하는 방법을 배웠습니다.

Project 01

GOAL

Responsive Web[PC, TABLET, MOBILE]

골(GOAL)은 주로 세계적으로 거대한 규모를
가진 축구의 모든 정보들을 볼 수 있는 브랜드이자 웹사이트입니다.
기존에 있는 골사이트를 새롭게 리뉴얼하고 축구의 정보를
소개하는 반응형 웹사이트로 제작을 했습니다.

Project 04

One Page Web

우리나라 커피브랜드인 카누 커피를
